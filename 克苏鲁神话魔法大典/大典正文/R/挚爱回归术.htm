挚爱回归术〔造〕

Return of the Beloved

消耗：可变的魔法值；5 点 POW；1D10 点理智值
施法用时：1 天

创造刚刚死亡的人的短暂幻影。法术需要举行一整天的仪式，施法者消耗和他想要幻影存留天数相同的魔法值。要复制的人必须死亡不超过一星期。这个幻影的所有细节与原本一般无二，但只有原本一半的 INT，而且每句话只说一个字。他记得原本的所有人际关系和技能，但知识类技能（如「考古学」等）都会减半。他也不太可能施放原本懂得的法术（使用新 INT 值，要困难 INT 检定成功才能回想一个法术）。

法术终止后幻影就会倒在地上化为一缕轻烟。复生死者并恢复意识可能惊扰原本的朋友和家人，造成理智损失（损失 1/1D6+1 点）。

别名：死者的幻影、重构术、死者的恩惠
