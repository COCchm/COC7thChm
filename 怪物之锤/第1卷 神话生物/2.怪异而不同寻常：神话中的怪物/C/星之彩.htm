星之彩

这是一束磷光，太空群星之间的奇异之光——有些人称它为坠落星辰。它徘徊在降落之处，渗透入土地和水中，感染地球上的生命，使之恶化。这种溪水般流动的、怪异的、有感知能力的色彩，因与自然界的任何事物没有相似之处而无法归类。它并非气态，而是无形；当它移动时，有人说它清晰可见，像不定形的闪耀光斑，在对人类双眼而言陌生的、深邃而不自然的光谱内，以诸多色度翻滚闪烁着。它进食时，其受害者的皮肤和面部会散发同样色彩的微光，在它影响范围内的动植物也会这样。它并非物质实体，但被它经过的感觉就像接触了湿滑黏稠的蒸汽一般。

别名：幻彩之色，太空种子，怪异访客，流星之卵。

星之彩来自于自然法则截然不同的空间或其它的维度。成年星之彩会产生搭载着胚胎的球体，它们被送到宇宙中旅行，在其它的世界寻找营养。倘若球体落在植被茂密的土壤或者浅水之中，胚胎就会开始生长。在几天之后，外壳溶解，全新的个体就此诞生，我们可以称之为幼体。胶状的幼体可以长到很大。在它渗透并感染生态系统时，当地的植被会发生大规模而不健康的成长。果实膨大，收成丰茂，但此时得到的农产品尝起来却是苦涩而令人不快的。此地出生的昆虫和动物将会产生畸形或者突变。尽管在白天几不可见，在夜里，被星之彩感染的地方特征十分明显，因为所有的植物都会发出色彩怪异的微光，草木开始扭曲蠕动，仿佛处在强风之中。甚至那些在星之彩所感染的地方生活的人类也会发出同样波长的光。

在几个月之后，幼体将成长成幼年星之彩。它此时会短暂地离开巢穴觅食，并开始吸取这片地区的生命力——字面意义上吮干这片土地和一切生活在上面的生命。对人类来说，这种生命吸取表现为受害者的皮肤老化、变灰和开裂，面部出现凹陷，奇怪的肉质增生也可以作为证据。一旦被吸干，被感染的人就会死去，动植物也是如此。一只星之彩可能会吸干大约 5 英亩 ( 大约 20,000 m2 ) 的地区。这个地区将会就此毁灭，再没有植物可以生长其中。在吸取了足够的能量后，成熟的星之彩便会离开这颗行星，于星空中步入成年。

检测：盖革计数器也许会把星之彩的存在记录为显著的辐射爆发。加装现代的光强化装置可以将之记录为一束强光，红外探测仪则毫无效果。

厌光：明亮的光线可以抑制星之彩活动；它会在阴暗、凉爽的藏身处度过白天，尤其是水下空间（蓄水池，水井，湖泊，水库和海洋都可以）。

分解：星之彩可以集中的能量，将几乎任何材料分解出一个洞。这个能力主要用于挖掘地下巢穴。用于融化一立方英尺 ( 23 升 ) 的钛的力量可以使几立方码 / 米的松木消失。以此法挖掘的孔洞各个侧面似乎都被熔化过，但星之彩并没有产生热量。

实体化：可以将自己的一部分聚集并实体化，这一部分会变得半透明。它随后可以影响实物。

战斗方式
每轮攻击次数：1 ( 特殊 )
由于星之彩作为袭击者是如此高效，守秘人也许希望能让调查员看到它的到来，侦查检定能察觉到细小的微光，或注意到突然出现的臭氧味道。星之彩缺乏可辨别格斗技能，且无法被普通手段伤害。

属性吸取：当星之彩进食时，受害者必须进行 POW 对抗检定；如果星之彩获胜，它永久地吸食受害者的 STR，CON，POW，DEX 和 APP 每一项各 1D10 点。吸取的每 1 点 POW 都会使星之彩的 POW 相应增加。受害者能够感受到被吮吸和灼烧的感觉，并逐渐地虚弱和老化。

精神攻击：可以削弱和腐化周围有感知能力的生物的心智。每在星之彩附近区域度过一天，每个人就必须进行一次 INT 对抗检定，失败则损失 1D6 点魔法值和 1D6 点理智值。除非离开这片地区，损失的魔法值不会恢复。这种影响也将强烈地将受害者困在该地区，随着受害者的意志减弱，该影响变得越来越不可抗拒。要离开这个地区，他们必须进行成功的困难或者极限 POW 检定(若魔法值减少到 50% 则困难难度，魔法值减少到 0 则极限难度)，或者便会难以解释地被迫留下。
