贴身男仆（Valet）

“人们不禁注意到，即使是那些“闲散的富人”很活跃，他们似乎也做不出什么了不起的事情。当然，并不是每个富人都一样，但有些人是如此的无知，以至于他们认为白色晚宴夹克是适合晚上穿的服装。想象一下，一个绅士需要有人引导他处理生活中遇到的小小复杂事情。他必须在心灵和钱包的问题上受到教育，可以说。他必须时刻保持警惕。一个草率的求婚，一个不幸的赌注，或者一个鲜红的腰带，都可能毁掉一个绅士的名声。他们受过良好的教育，却并不聪明。我真的不认为他们中的很多人能够独自生活。毕竟，当一个人连今天是星期几都不知道的时候，要保持自己的社交日程有序，往往是很困难的。”

贴身男仆是“绅士的私人绅士“。他们做饭、开车、安排行程、保证其主人无可挑剔，更确切地说，发挥各种社交功能。贴身男仆之间的竞争并不少见。能保证自己的主人被精心照料、毫无瑕疵往往是一件值得骄傲的事。当然这样的竞争只会通过无害的竞争表现出来，贴身男仆会做一些小破坏来让“他们的”主人在上流社会中脱颖而出。比如一个消失的袖扣盒157可能意味着一场大灾难。由于他们的职责，贴身男仆通常是未婚的，他们或许在一战中曾与身为士官158的主人一起服役，也可能单独服役。

贴身男仆必须头脑冷静、富有洞察力、善于应对各种工作。最重要的是，他必须能够在最糟的情况下使用任何必要的手段拯救他的主人：一杯醉人的鸡尾酒、一剂醒酒药、对更适当行为的礼貌暗示，如果有必要的话，也可以是配枪中射出的一两枚精准命中的子弹。绅士们总有一种在最离奇的情况下给自己惹麻烦的倾向，在无计可施的时候，韦伯利 MK.VI159胜于所有口舌。

贴身男仆是一个多样的团体。如果他们是退役军人，他们的 STR, CON，和 DEX 将等于或高于平均值。所有贴身男仆为了“激励”自己的主人都会有较高的 INT 和 POW。他们的 APP 可能等于或低于平均值，特别是当他们有拳击的倾向时。EDU 和 SIZ 可能等于或略低于平均值。

本职技能：会计，手艺（厨艺），汽车驾驶，礼仪，话术，射击（手枪），射击（步枪/霰弹枪），机械维修。贴身男仆的信用等级不应高于 40。
推荐个人兴趣技能：格斗（拳击），魅惑，电气维修，急救，聆听，骑术，潜行，游泳，投掷，追踪。
信用评级：20~40
本职技能点数：EDU x2 + DEX x2 或 STR x2

157：袖扣是用于固定正装衬衫袖口的珠宝首饰，仅适用于袖口两侧有扣眼但没有纽扣的衬衫。1920 年代，佩戴的袖扣有各种形式、颜色和材料可供选择，在更便宜的副本中包括宝石和宝石较少的宝石和玻璃。各种几何图案的复杂彩色珐琅袖扣特别受欢迎。
158： NCOs 是 non-commissioned officers 的缩写，士官。
159：Mark VI Webley，是韦伯利转轮手枪中的最著名的型号之一。
